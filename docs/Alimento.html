    Descripción¶ ↑

    La clase alimento fue creada para representar un objeto alimento con las siguientes características:

    • Proteínas

    • Hidratos de carbono

    • Lípidos

    • Gases de efecto invernadero

    • Terreno usado

    new(name, prot, carb, lip, co2, terr, gr) click to toggle source

    Constructor de la clase. Dentro de él, además, se calcula los atributos por porción de alimento. Para ello, se divide el atributo entre 100g (datos provenientes de la tabla dada en la Práctica) y se multiplica por los gramos reales de alimento.

    • Name(tipo string): Nombre del alimento

    • Prot(tipo double): Proteínas por 100g

    • Carb(tipo double): Carbohidratos por 100g

    • Lip(tipo double): Lípidos por 100g

    • C02(tipo double): Gases de efecto invernadero producidos por 1kg

    • Terr(tipo dobule): Metros cuadrados de terreno usado

    • Gr(tipo double): Cantidad de gramos de alimento

    # File lib/Practica6/alimentos.rb, line 46
def initialize(name, prot, carb, lip, co2, terr, gr)

    @nombre = name
    @proteinas = (prot / 100.0 * gr).round(3)
    @carbo = (carb / 100.0 * gr).round(3)
    @lipidos = (lip / 100.0 * gr).round(3)
    @gei = (co2 / 100.0 * gr).round(3)
    @terreno = (terr / 100.0 * gr).round(3)
    @gramos = (gr).round(3)
  
end

def to_s
  
    os = ""
    os += "Alimento: #{nombre}" + "\n"
    os += "Cantidad de gramos(racion): #{gramos}" + "\n"
    os += "Cantidad de gramos de proteinas(en 1kg): #{proteinas}" + "\n"
    os += "Cantidad de gramos de carbohidratos(en 1kg): #{carbo}" + "\n"
    os += "Cantidad de gramos de lipidos(en 1kg): #{lipidos}" + "\n"
    os += "Cantidad de GEI(kgCO2eq): #{gei*100.0}" + "\n"
    os += "Cantidad de terreno(m2año): #{terreno*100.0}" + "\n"
    
    os
end
