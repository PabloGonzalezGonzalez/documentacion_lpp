    Descripción¶ ↑

    La clase lista implemente una lista doblemente enlazada. Usa nodos y posee:

    • Head: cabeza de la lista

    • Tail: cola de la lista

    • Size: tamaño de la lista

    Clase nodo¶ ↑

    Estructura creada para ser las unidades de la lista. Contiene:

    • Dato: Objeto dentro del nodo

    • Sig: Nodo siguiente

    • Ant: Nodo anterior

def delete_head

def each
    i = @head
    f = @size
    k = 0
    while (k < f) do
        yield i.dato
        i = i.sig
        k = k+1
    end
end

def insert_head(node)
    if (node.is_a?Struct::Nodo)
        if (@size == 0)
            @head = node
            @tail = @head
        else
            node.ant = Struct::Nodo.new(nil,nil,nil)
            node.sig = @head
            @head.ant = node
            @head = node       
        end
    else
        aux = Struct::Nodo.new(node,nil,nil)
        if (@size == 0)
            @head = aux
            @tail = @head
        else
            aux.ant = Struct::Nodo.new(nil,nil,nil)
            aux.sig = @head
            @head.ant = aux
            @head = aux  
        end
    end
    @size = @size + 1

end

def to_s
    os = ""
    if (@size == 1)
        os += @head.dato.nombre + "."
    else
        i = 0
        nodo_actual = @head
        while (i < size)  
            if (i == size - 1)
                if (!nodo_actual.nil?)
                    os += nodo_actual.dato.nombre + "."
                end
            else
                if (!nodo_actual.nil?)
                    os += nodo_actual.dato.nombre + ", "
                end
            end
            if (nodo_actual.sig != nil)
                nodo_actual = nodo_actual.sig
            end
            i = i + 1
        end
    end
    os
end
